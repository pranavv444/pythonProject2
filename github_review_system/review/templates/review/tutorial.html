Setup Guide

Step 1: Access Your GitHub Repository

Navigate to the GitHub repository where you want to set up the PRReview.ai automation. Ensure you have the necessary permissions (typically admin rights) to add webhooks to this repository.

Step 2: Open Webhooks Settings

In your repository, click on the "Settings" tab located in the top right of the page. Scroll down the sidebar on the left and select "Webhooks".

Step 3: Add New Webhook

On the Webhooks page, click the "Add webhook" button to create a new webhook for PRReview.ai.

Step 4: Enter Payload URL

In the "Payload URL" field, paste the URL provided by PRReview.ai. This URL will receive the webhook data from GitHub.

Step 5: Select Content Type

Under "Content type", select "application/json". This setting ensures that GitHub sends JSON-formatted data to your webhook URL, which is the standard format and easily processed by our AI model.

Step 6: Choose Events to Trigger the Webhook

Scroll down to the "Which events would you like to trigger this webhook?" section. Select "Let me select individual events". Then, scroll down and check the boxes for:

  • Pushes
  • Pull requests
  • Pull request reviews

At the bottom, ensure the "Active" checkbox is selected to enable the webhook.

Finally, click the "Add webhook" button to save your configuration.
